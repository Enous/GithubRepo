# 🌌 EnpaiManage (Enpai Dev)

**EnpaiManage**, GitHub repolarınızı tek tıkla organize eden, yapay zeka destekli şık ve pratik bir yönetim aracıdır.

---

## 🚀 Temel Özellikler

- **🤖 AI Kod Analizi**: Groq Cloud entegrasyonu ile indirilen repoların ne işe yaradığını saniyeler içinde analiz edin. (Llama 3.3-70B desteği!)
- **📂 Akıllı Kategorizasyon**: Repoları otomatik olarak AI, Web, Security, Python gibi klasörlere ayırır.
- **🔍 Canlı Arama & Filtreleme**: Koleksiyonunuzdaki repoları isme veya kategoriye göre anında bulun.
- **🔄 Canlı Senkronizasyon**: Klasörden sildiğiniz bir dosya uygulamadan da otomatik olarak silinir.
- **📝 Profil README Oluşturucu**: Koleksiyonunuzu şık bir Markdown listesi olarak dışa aktarın ve GitHub profilinizde paylaşın.
- **💻 VS Code & Git Entegrasyonu**: Repoları tek tıkla VS Code'da açın veya `git pull` ile güncelleyin.
- **💾 Boyut Takibi**: Her reponun diskte ne kadar yer kapladığını görün.

---

## 🛠️ Başlangıç

1.  **Kurulum**: `kurulum.bat` ile gerekli kütüphaneleri (PyQt6) yükleyin.
2.  **Başlat**: `baslat.bat` ile uygulamayı açın.
3.  **Yapay Zeka Analizi**: Kodları analiz etmek için [Groq Cloud](https://console.groq.com/) üzerinden ücretsiz bir API key alıp Ayarlar kısmına yapıştırın.
4.  **Kullanım**: GitHub linkini yapıştırın ve **İndir** butonuna basın.

---

## 🔒 Güvenlik ve Yetki
Uygulama, Windows üzerindeki kilitli Git dosyalarını yönetebilmek için **Yönetici yetkisi** ile çalışır. Tüm verileriniz (Token, Key vb.) yerel bilgisayarınızda (`~/.config/enpaimanage/`) güvenli bir şekilde saklanır.
